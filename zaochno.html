Первый Второй Третий

Базовый курс «Экспресс заочный». Порядок обучения

  • Курс длится 2 недели.
  • Вы изучаете продукты по видео-урокам по расписанию. Расписание опубликовано ниже.
  • Раз в несколько дней предусмотрено подключение через площадку ZOOM для возможности задать вопросы по пройденному материалу.
  • Для всех занятий создана единая ссылка для подключения.
  • Находится в закреплённом сообщении в Телеграм-канале.
  • В течение курса предусмотрены домашние задания для подготовки к экзамену.
  • Cдача всех тестов домашних заданий является допуском к экзамену.
  • Тесты домашних заданий находятся в разделе «Тесты».
  • Методические материалы в электронном виде находятся в разделе каждой лекции. Кнопка «Материалы»
  • В течение курса предусмотрено несколько организационных встреч с куратором через площадку ZOOM
  • Все вопросы можно задать в чате группы в Телеграм-канале.

Текущая группа: 1121Э.
